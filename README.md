# GoPro filters

This repo provides a [`list of filters`][filtersList] that could be applied to a media. 

`Filter` example

```json
{
    "name": "waimea",
    "rate_range": [
      0.0,
      1.4
    ],
    "categories": [
      "outdoor",
      "vintage",
      "colors"
    ],
    "thumbnail_url": "https://raw.githubusercontent.com/nduponchel-gpsw/GoProFilters/main/thumbnails/duke.png"
}
```
